# Unicup generated asset pack

Procedurally generated original placeholder/polish assets prepared for integration.

- `abilities/*.png`: 256x256 ability icons for every box type plus mystery box.
- `abilities/ability-spritesheet.png`: 4x3 sprite sheet, order matches manifest.
- `sprites/ball-texture.png`: ball texture reference.
- `sprites/goal-gates.png`: blue/red gate sprite sheet.
- `sprites/field-props.png`: bumper/ramp/boost prop reference sheet.
- `manifest.json`: stable paths for the renderer/UI.

Meshy API key was not found in `/Users/mac/projects/personal` by key-name/content search, so no remote 3D generation was performed.
