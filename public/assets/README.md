# Unicup generated asset pack

Original generated and procedural assets prepared for the live renderer.

- `abilities/*.png`: 256x256 ability icons for every box type plus mystery box.
- `abilities/ability-spritesheet.png`: 4x3 sprite sheet, order matches manifest.
- `sprites/ball-texture.png`: ball texture reference.
- `sprites/goal-gates.png`: blue/red gate sprite sheet.
- `sprites/field-props.png`: bumper/ramp/boost prop reference sheet.
- `teams/*-robot.webp`: four original Unicup robot portraits used by the roster and team picker.
- `robots/*-surface.jpg`: generated high-resolution metal surfaces applied directly to each 3D chassis.
- `maps/*-field.jpg`: orthographic court textures used by the Three.js field plane.
- `maps/*-surroundings.jpg`: arena panoramas used behind each map.
- `abilities/readPlay.png`: Read the Play opponent-intent vision icon.
- `manifest.json`: stable paths for the renderer/UI.
